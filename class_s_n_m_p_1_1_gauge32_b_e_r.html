Detailed Description

BER object to handle 32-bit gauge.

  • Type is TYPE_GAUGE32.
  • Length is variable, from 1 to 5 byte(s), short form encoding.
  • Size is variable, from 3 to 7 bytes.

Example

Gauge32 Encoding
0 41 01 00
4294967295 41 05 00 FF FF FF FF
