Detailed Description

BER object to handle octet string.

  • Type is TYPE_OCTETSTRING.
  • Length is variable.
  • Size is variable.

Example

Octet String Encoding
01 23 45 67 89 AB CD EF 04 08 01 23 45 67 89 AB CD EF
